Uninstalling ROS¶

If you’re running Ubuntu, and installed ROS with apt-get, the first step would be:

sudo apt-get remove ros-*

Remove configuration files:

sudo apt-get purge ros-*

Remove packages that are no longer required:

sudo apt autoremove
