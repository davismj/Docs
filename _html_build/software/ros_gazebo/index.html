Examples:¶

Get the position of a model:

gazebo::math::Pose pose = this->model->gazebo::physics::ModelState::GetPose();

Pose to initialize the model to:

msgs::Set(msg.mutable_pose(),
math::Pose(math::Vector3(1, -2, 0), math::Quaternion(0, 0, 0)));
